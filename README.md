# Maazxp Portfolio Website

موقع شخصي لعرض مشاريعي ومهاراتي في تطوير الألعاب باستخدام محرك Godot.

## المميزات
- تصميم متجاوب يعمل على جميع الأجهزة
- دعم اللغتين العربية والإنجليزية
- وضع ليلي ونهاري
- تأثيرات تفاعلية سلسة
- تحسينات SEO متقدمة

## التقنيات المستخدمة
- HTML5, CSS3, JavaScript
- Godot Engine, GDScript
- Blender, Photoshop
